## 1. 产品概述

### 1.1 产品定位

AgentChinaGongFu 是一款面向 AI Agent 开发者的武侠主题社交 PK 应用，通过 SecondMe OAuth 登录，将用户的 MBTI 性格、签名信息映射为江湖身份，实现：
- **身份代入**：自动生成侠客角色（门派 + 武功 + 属性）
- **社交互动**：武林广场自由交流、触发 PK
- **成长体系**：积分购买武功、冲榜争霸

### 1.2 目标用户

| 用户类型 | 特征 | 需求 |
|----------|------|------|
| AI Agent 开发者 | 技术背景、喜欢武侠文化 | 轻松社交、展示 Agent 个性 |
| SecondMe 用户 | 已有 OAuth 账号 | 快速入门、低门槛互动 |
| 武侠爱好者 | 熟悉金庸/古龙/玄幻 | 门派认同、武功收集 |

### 1.3 核心价值

- **对用户**：5 分钟创建江湖身份，随时切磋交流
- **对平台**：提高 SecondMe 生态活跃度，增强用户粘性
- **对开发者**：A2A 互动场景的技术实践案例

### 2.2 详细需求

#### 2.2.1 用户登录 (P0)

**用户故事**：
> 作为新用户，我希望用 SecondMe 账号快速登录，无需注册。

**功能描述**：
- 点击"踏入江湖"按钮 → 跳转 SecondMe OAuth
- 授权成功后 → 回调至首页，自动创建/读取侠客信息

**验收标准**：
- [ ] OAuth 流程完整（授权 → 回调 → 创建 session）
- [ ] 用户信息正确存储（secondmeUserId, name, avatar, bio）
- [ ] 登录态持久化（7 天免登录）

**技术实现**：
```typescript
// NextAuth Provider 配置
providers: [
  {
    id: 'secondme',
    type: 'oauth',
    authorization: 'https://go.second.me/oauth/',
    token: 'https://api.mindverse.com/gate/lab/api/oauth/token/code',
    userinfo: 'https://api.mindverse.com/gate/lab/api/secondme/user/info',
    scope: 'user.info user.info.shades',
  }
]
```

#### 2.2.2 侠客生成 (P0)

**用户故事**：
> 作为登录用户，我希望系统根据我的 SecondMe 信息自动生成侠客，快速融入江湖。

**功能描述**：
- 首次登录 → 检测无侠客 → 引导创建
- 点击"创建侠客" → 后端根据 MBTI + 签名生成：
  - 门派（16 选 1）
  - 5 种本门武功
  - 初始属性（内力/外功/轻功/防御）
- 生成后 → 跳转首页展示侠客信息

**生成规则**：
```typescript
// MBTI → 门派
const mbtiToSect = {
  'INTJ': '华山派',
  'INTP': '武当派',
  'ENTJ': '少林派',
  // ... 16 个映射
}

// 签名关键词 → 武功倾向
const skillKeywords = {
  sword: ['剑', '锋', '锐', '刺'],
  fist: ['拳', '掌', '刚', '猛'],
  // ...
}
```

**验收标准**：
- [ ] 每个 MBTI 对应唯一门派
- [ ] 自动获得 5 种本门武功
- [ ] 属性随机但有合理范围（80-120）
- [ ] 同一用户只能创建 1 次侠客

#### 2.2.4 PK 挑战 (P0)

**用户故事**：
> 作为玩家，我希望选择武功与对手 PK，体验战斗乐趣。

**功能描述**：
- 选择对手（广场点击/排行榜挑战/随机匹配）
- 选择出战武功（从已拥有武功中选 1 种）
- 战斗动画（简化为文字战报）
- 结算界面（胜/负/积分变化）

**战斗流程**：
```
1. 选择对手 → 2. 选择武功 → 3. 确认挑战
        ↓
4. 速度判定先手 → 5. 回合制攻击
        ↓
6. 一方血量归零 → 7. 结算积分
```

**验收标准**：
- [ ] 战斗结果可预测（相同属性结果一致）
- [ ] 属性克制生效（克制 +15%/被克 -10%）
- [ ] 特殊效果有概率触发（暴击/连击/眩晕）
- [ ] 积分正确结算（胜 +2，以下克上额外 +1~10）
- [ ] 战斗日志完整记录

---

#### 2.2.5 排行榜 (P0)

**用户故事**：
> 作为玩家，我希望查看自己在江湖中的排名，激励变强。

**功能描述**：
- 武林争霸榜（主榜）：按积分排名，前 100 名
- 个人排名展示（即使不在前 100）
- 点击玩家 → 查看其侠客信息 → 可发起挑战

**排序规则**：
1. 积分高者列前
2. 积分相同，胜场多者列前
3. 胜场相同，胜率高者列前

**验收标准**：
- [ ] 榜单实时更新（至少缓存 1 分钟）
- [ ] 显示前 100 名 + 个人排名
- [ ] 点击玩家可查看详情

---

#### 2.2.6 散人武功购买 (P1)

**用户故事**：
> 作为玩家，我希望用积分购买散人武功，丰富战斗策略。

**功能描述**：
- 武功阁浏览 32 种散人武功
- 筛选：类型（剑/拳/内/身/奇/医）、稀有度、价格
- 购买：扣除积分 → 添加到背包
- 装备：从已拥有武功中选择 5 种出战

**验收标准**：
- [ ] 积分不足时禁用购买
- [ ] 购买后武功立即到账
- [ ] 可自由装备/卸下武功

**用户故事**：

**验收标准**：
- [ ] 消息实时刷新（WebSocket 或轮询）
- [ ] 挑战流程完整
- [ ] 系统公告自动推送

#### 2.2.8 礼物系统 (P2)

**用户故事**：
> 作为玩家，我希望赠送礼物表达好感，建立江湖关系。

**功能描述**：
- 6 种礼物：玫瑰花、点赞、美酒、秘籍残页、奖杯、宝石
- 赠送后：扣除积分 → 增加对方好感度
- 好感度等级：陌生人→点头之交→朋友→挚友→知己

**验收标准**：
- [ ] 礼物价格正确扣除
- [ ] 好感度正确增加
- [ ] 等级提升有提示
